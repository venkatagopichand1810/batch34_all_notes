# Portfolio Website

A simple portfolio website built with React and Vite, showcasing skills, projects, and contact information for a fresher developer.

## Features

- **About**: Introduction and background
- **Skills**: List of technical skills
- **Projects**: Showcase of sample projects
- **Contact**: Contact details and links

## Technologies Used

- React
- Vite
- CSS

## Getting Started

1. Clone the repository
2. Navigate to the `client` directory
3. Install dependencies: `npm install`
4. Start the development server: `npm run dev`
5. Open [http://localhost:5173](http://localhost:5173) in your browser

## Build for Production

Run `npm run build` to build the app for production.
